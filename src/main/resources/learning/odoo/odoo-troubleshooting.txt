Database performance issues:

The Odoo database (PostgreSQL) might be experiencing slowdowns due to lack of indexing, fragmentation, or a growing dataset
Long-running queries might be blocking other operations


Disk I/O bottlenecks:

Slow disk access or high disk I/O wait times can cause application slowness even with low CPU/RAM usage
Check if your disk is becoming full or if there are I/O wait times in your monitoring


Network latency:

If Odoo components are distributed across different servers, network issues could cause slowdowns
Check for connectivity issues between application and database servers


Specific Odoo processes:

Some operations like report generation, PDF creation, or heavy calculations might be single-threaded
Large batch operations might be consuming specific resources


Connection pooling:

Too many open database connections might be causing contention
